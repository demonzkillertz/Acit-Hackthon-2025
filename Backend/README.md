# Bus Tracking & Passenger Safety Backend

Node.js backend for real-time bus tracking, passenger safety, and fleet management.

## Features
- User, driver, and company authentication
- Live bus tracking (GPS)
- Route visualization (Leaflet-ready)
- Alerts and notifications
- Reviews, favorites, and safety features
- PostgreSQL database

## Setup
1. Copy `.env.example` to `.env` and fill in your values.
2. Install dependencies: `npm install`
3. Run in dev mode: `npm run dev`
